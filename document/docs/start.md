## Loading Dependencies

DeepMASS may take some time to automatically load the necessary data. 
Please be patient and wait until the buttons become active.

[![sceenshot-1.png](https://i.postimg.cc/1RpSD19f/sceenshot-1.png)](https://postimg.cc/jCdBr9nb)

## Opening a File

Click the **Open** button and select an *mgf*, *msp* or *mat* file containing one or multiple MS/MS spectra. 
You can find an example file here. [Example here](https://github.com/hcji/DeepMASS2_GUI/blob/main/example/all_casmi.mgf)

[![sceenshot-2.jpg](https://i.postimg.cc/fys68kwk/sceenshot-2.jpg)](https://postimg.cc/N9NJMs2w)

## Running Analysis

Choose either **Run DeepMASS** to annotate spectra using the DeepMASS algorithm or **Run MatchMS** for 
library matching. Wait for the progress bar to complete the analysis.

[![sceenshot-3.jpg](https://i.postimg.cc/vHQJ91qN/sceenshot-3.jpg)](https://postimg.cc/K4HphYyP)

## Saving Results

Click the **Save** button and select the folder path where you want to save the annotation results.

[![sceenshot-4.jpg](https://i.postimg.cc/50BrQ1wJ/sceenshot-4.jpg)](https://postimg.cc/FdKpwMWC)
